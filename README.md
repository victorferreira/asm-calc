# asm-calc
---
## Description
A calculator implemented in assembly.

---
## Reserved Symbols

| Symbol | Description    |
| -------|:--------------:|
| +      | addition       |
| -      | subtraction    |
| *      | multiplication |
| /      | division       |
| ^      | exponentiation |
| =      | attribution    |

---
## Syntax rules
- Variables must start with letters
- Every statement must start on a new line

---
## Examples of a valid input

```calc
a = 12 * 3
2 + 1
b = 3 ^ a
```
